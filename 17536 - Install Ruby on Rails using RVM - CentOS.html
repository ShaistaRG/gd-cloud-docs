Steps to install Ruby on Rails using Ruby Version Manager (RVM) in CentOS, Ruby, Rails

Install Ruby on Rails using RVM – CentOS

Difficulty: 1

Time: 30 minutes

Ruby on Rails® is a well-known open source stack for web application development. The Ruby Version Manager (RVM) lets you easily manage multiple Ruby application environments and switch between them in a most efficient way. This article will assist you on how to setup Ruby on Rails in CentOS using RVM.

Prerequisites

  • To initially setup the server, please visit 17323 - Initial Server Setup - CentOS

You will need access to sudo privileges to complete these steps.

Prepare your system

Before you can install Ruby on Rails, prepare your system with the required and updated system software.

  1. Access root by running the following command:
    sudo yum update
  2. To install curl use following command.
    sudo yum install curl
  3. Set up the mpapis public key with the following command:
    gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3

Install RVM

  1. If your system doesn’t have RVM already installed, use the following command to install RVM.
    \curl -L https://get.rvm.io | bash -s stable --ruby

    The backslash before curl prevents potential version conflicts and --ruby flag will install the latest version of Ruby.

  2. Following message will appear after all installation process.
    To start using RVM you need to run `source /home/user_name/.rvm/scripts/rvm` in all your open shell windows, in rare cases you need to reopen all shell windows.
    To apply RVM, execute the following command:
    source /home/user_name/.rvm/scripts/rvm

    If your system already has RVM installed, update it, and install Ruby using the following commands.

    rvm get stable --autolibs=enable
    rvm install ruby

    If you’ve already installed these packages, you'll get a message

  3. Rail asset pipeline needs JavaScript runtime to compile code. On CentOS, Node.js (a server-side JavaScript environment) is preferred. To install this, type the following command:
    sudo yum install nodejs

Install Gems

Bundler is a tool to manage gems while developing and running Rails applications. Nokogiri is a dependency for many other gems.

  1. Check the version of gem manager.
    gem –v
    To update gem manager type following command.
    gem update –-system
  2. Use Ruby with gemset global.
    rvm gemset use global
  3. Install Bundler and Nokogiri gems with the following commands:
    gem install bundler
    gem install nokogiri

Install Rails

  1. Use gem to install the most recent, stable version of Rails.
    gem install rails
  2. To check the version, type:
    rails –v
    To install a specific version, type:
    gem install rails --version=[version number]

After following all these steps, you are done with installation of Ruby on Rails.
