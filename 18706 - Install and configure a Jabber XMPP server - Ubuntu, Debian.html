Configuring Jabber XMPP server in Debian and Ubuntu, XMPP

Install and configure a Jabber XMPP server - Debian,Ubuntu

Difficulty: 2

Time: 20 Minutes

Jabber is simple, mature and powerful massaging protocol based on XMPP developed for real-time text massaging between two or more people. Jabber is used by enterprises for internal communication, by developers to improve Erlang programming skill, by hosting provider for highly stable communication services,by individual for chatting fun. Jabber is written in Erlang . It has its own web-based management interface. This article explains the installation of Jabber XMPP server in Debian and Ubuntu.

You'll need access to root privileges to complete these steps.

Install Jabber

  • Use following command to install the Jabber server along with its components.
    sudo apt-get install ejabberd

You have installed the server, but we need to change a few configuration settings.

Configure Ejabberd

  1. Its configuration file reside in /etc/ejabberd/.Open ejabberd.cfg from this directory.
    sudo vim /etc/ejabberd/ejabberd.cfg
  2. To add the host name you want server to use. Update this line:
    {hosts, ["localhost"]}

    with this line:

    {hosts, ["localhost","coolexample.com"]}
  3. To create an admin user. Update this line:
    {acl, admin, {user, "", "localhost"}}

    with this line:

    {acl, admin, {user, "my_admin", "coolexample.com"}}
  4. Save and close the file by pressing esc and then type following character followed by pressing Enter.
    :wq
  5. Restart the service.
    sudo service ejabberd restart
  6. To set password for admin user, use following command:
    sudo ejabberdctl register my_admin coolexample.com password

Next steps

After following these steps you will have overview of installation and basic configuration of jabberd server. To change any other configurations log into the web-based management panel. In your browser, go to http://coolexample.com:5280/admin and enter admin credentials.
