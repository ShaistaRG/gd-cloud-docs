Basic Linux Navigation and File Management - ALL DISTROS

Difficulty: 1
Time: 15 minutes

In this article, we will assist you to understand the basic Linux navigation and file management in all distros. To perform this task, we are assuming that the user has sudo privileges.

Find where you are with the pwd command

  1. To know the present location of any directory
    pwd

    The result should look like this

    /home/username

Look at the content of directories with ls

  1. To change the directory
    cd /usr/share
  2. To list all the directories and files
    ls

    The output will be shown as

  3. To know all the information about permissions of all directories and files
    ls –l

    The output will be shown as

  4. To get a listing of all files, including hidden files and directories
    ls -al

View Files

  1. To view a particular file, use the following command
    less /etc/services

    The output will be shown as

    If you want to scroll on the output file, then press the up and down arrow keys on the keyboard. To scroll one page down the screen, press either CTRL-f, space bar or the ‘Page Down’ button on the keyboard. To scroll one page up the screen, press either CTRL-b or ‘Page Up’ button on the keyboard.

  2. If you want to search something like mail in this file, then type
    /mail
  3. To exit
    /q
  4. To display the content of a particular file
    cat file1
  5. To display top 10 lines of a particular file
    Head /etc/services
  6. To display last 10 lines of a particular file
    Tail /etc/services

Working with Files and Directories

  1. Many commands can create files. You can use the touch command to create a file. This command will create an empty file with the name and location that you specify
    sudo touch file1
  2. We can create multiple files at the same time using the following command
    sudo touch file1 file2 file3 file4
  3. To create a directory
    sudo mkdir test
  4. To create nested directories
    sudo mkdir –P /test/share/example
  5. To rename directories and file using
    sudo mv test share //for directories
    sudo mv file1 example1 // for files
  6. To copy directories and files
    sudo cp -rvf test again // for directories
    sudo cp //source path //destination path

    For example

    sudo cp –rvf /etc/services /use/file1

    Here -r means recursive
    -v means visible
    -f means forcedly

  7. To remove directories and files
    sudo rm –rvf /etc/services.txt // for files
    sudo rmdir –rvf /etc/test //for directories

    Here -r means recursive
    -v means visible
    -f means forcedly

Edit Files

  1. Open a file
    sudo vim /etc/services
  2. Press “I” to go in insert mode and make changes whatever you want.
  3. Press “ESC” to go in maintenance mode.
  4. To save the file
    :wq!
  5. To exit without saving the file
    :q!

Conclusion

In this article, we reviewed the steps involved with the basic Linux navigation and file management in all distros.
