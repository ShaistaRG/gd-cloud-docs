What are MySQL and MariaDB?

MySQL and MariaDB are relational databases. These tools are used to handle large volumes of data for different kind application programs. Both uses SQL query language to retrieve the information from different tables and databases. Both MySQL and MariaDB support Virtual Private Server (VPS) or cloud to manage your data.

Perquisites

In this article, you can use Ubuntu (or any other distros), MariaDB and MySQL. MariaDB and MySQL can be used to query database for various filters or search criteria.

Create a basic query in MySQL and MariaDB

To select some information for the users:

SELECT user,password,host FROM user;

The above query will show result for the three fields which you have selected in the SELECT query. In this query, each of the fields are separated with a comma.

Wildcard usage in MySQL and MariaDB queries

By using the SELECT query in the above example, you can use specific fields to retrieve data. If you want to return all the information from the "user" table, then you can use asterisk '*'.

SELECT * FROM user;

Filter result in MySQL and MariaDB

You can apply filter for results that you want should meet a specific criteria by using the where keyword. In the example given below, if you want the query to show user, password and host that has a value set as "root" for user, then use the following query:

SELECT user,password,host FROM user WHERE user = "root";

Same way you can apply filter for different fields. In the given example below, the result will show user, password and host from the user table which has a value named localhost for host:

SELECT user,password,host FROM user WHERE host = "localhost";

Like comparison operator

The "Like" operator compares the value from the right side of the expression, in this example l%. The below query will fetch user, password and host fields from user table where the criteria host is LIKE l%. So, this will result in an output with all the respective fields starting with letter "l".

SELECT user,password,host FROM user WHERE host LIKE "l%";

Here you can also apply and or or for equality comparison.

SELECT user,password,host FROM user WHERE host LIKE "l%" or host LIKE "127%";

The above query will match each record and return result with filter condition that matches the criteria i.e host starting from either l or 127.
